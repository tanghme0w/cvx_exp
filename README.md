# cvx_exp

### How to use

Create a directory named 'data' in the repository. Put all experiment materials inside it.

Tree:
```text
cvx_exp
├── data
│   ├── A_1.mat
│   ├── A_2.mat
│   ├── A_3.mat
│   ├── corrupted1.png
│   ├── corrupted2.png
│   ├── corrupted3.png
│   ├── figure1.png
│   ├── figure2.png
│   └── figure3.png
├── inpaint.py
└── show_recovered.py
```
